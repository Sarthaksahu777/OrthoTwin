MRI Browser Demo

Early-phase browser viewer for lumbar spine MRI. This static demo uses curated slice exports. The full Streamlit app loads DICOM volumes locally.

View Mode

About

The original Phase 1 app was built with Streamlit and pydicom. It browses RSNA 2024 lumbar spine DICOM series, shows degeneration labels, similar cases, and supports L4–L5 annotation on sagittal series.

Full DICOM volumes are not included in the public GitHub repository. To run the interactive viewer locally, see docs/README.md.
